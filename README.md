# Note-app📝 

A simple and elegant Note‑Taking application built with **HTML**, **CSS**, and **JavaScript**.  
Users can write notes, save them, and delete them — with all notes stored in **localStorage**, so they remain even after refreshing the page.



## 🚀 Features

- Create and save notes  
- Delete notes  
- Notes persist using localStorage  
- Clean and responsive UI  
- Works on CodePen, GitHub Pages, Netlify, and any static hosting  



## 📂 Project Structure



note-app/ │ ├── index.html ├── style.css └── app.js




## 💻 Technologies Used

- **HTML5** – Page structure  
- **CSS3** – Styling and layout  
- **JavaScript (Vanilla)** – App logic and localStorage  

No frameworks required.
